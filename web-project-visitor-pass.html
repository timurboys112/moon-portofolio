Visitor Pass Digital System

Web Development Project

Visitor Pass Digital System is a web-based application designed to streamline guest registration and monitoring processes in a residential environment.

Project Overview
  • Client: Apartemen Eksekutif Menteng
  • Role: Web Developer
  • Duration: May 2025 – June 2025
  • Tech Stack: HTML, Bootstrap, PHP, MySQL
  • Status: Internal System Prototype
Problem

Manual guest registration caused inefficiencies, inconsistent data records, and delayed visitor check-in processes.

Solution

Developed a responsive web application that allows digital visitor pass creation, centralized guest data management, and mobile-friendly access.

Key Features
  • Guest registration form with responsive layout
  • Digital visitor pass generation
  • Admin dashboard for visitor data monitoring
  • Optimized performance for mobile devices
